# Next App Mantine Tailwind Template

This is a minimal template for a Next.js project with Mantine and Tailwind CSS.

## Development

Run the development server:

```bash
npm run dev
```

## More information

- [Mantine](https://mantine.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
